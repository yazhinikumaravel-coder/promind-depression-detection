# 🎓 ProMind: Student Depression Detection Platform

An AI-driven predictive intelligence system that utilizes a high-performance **CatBoost Binary Classifier** to proactively detect depression risk profiles in university students based on lifestyle, psychological metrics, and academic data.

## 🚀 Key Engineering Features
* **Advanced Classifier Engine:** Built with CatBoost to efficiently navigate non-linear behavior profiles across 100,000 data rows.
* **Responsive Client Interface:** An interactive, modern UI built with Streamlit designed with adaptive color gradients based on diagnostic assessment outputs.
* **Persistent Security Node:** Integrated with an internal SQLite framework managing protected login nodes and structural prediction logging logs.

## 📊 Analytics Schema Implemented
The model captures and maps continuous tracking vectors across key indicators:
* **Academic Stressors:** CGPA, Core Study Hours
* **Circadian & Physiological Health:** Rest Duration, Weekly Kinetic Workouts
* **Behavioral Dynamics:** Social Network Exposure, Subjective Stress Gradients
